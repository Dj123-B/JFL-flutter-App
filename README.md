## ğŸ›  Tech Stack & Architecture

This is not just a UI wrapper; it is a complete full-stack system.

### **Mobile Client (Flutter)**
*   **Language:** Dart & Kotlin (Bridging)
*   **Architecture:** MVVM / Clean Architecture
*   **State Management:** Provider
*   **Key Features:**
    *   Secure Authentication (JWT)
    *   Real-time data fetching
    *   Complex form validation for Loan/Insurance applications

### **Backend API (Node.js)**
*   **Language:** JavaScript
*   **Framework:** Express.js
*   **Database:** MongoDB
*   **Hosting:** Render
*   **Security:** BCrypt password hashing, Input sanitization
